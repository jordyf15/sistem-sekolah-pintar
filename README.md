# Smart Learning System

Smart Learning System is a web based learning management system that have features that assist learning and teaching activities between students and teachers of junior and senior high school. The main features are classes, materials, assignments, forum, scores, progress, announcements, and agenda. This project was made for my thesis at Bina Nusantara University.

## Live Demo on Firebase Hosting

https://sistem-belajar-pintar-49f3f.web.app/

### Thread Detail Page

This is the thread detail page for both teachers and students. In this page, teachers and students can see the thread and it's replies. For the thread and each reply, the teachers and students can download it's attachments by clicking the download icon in thread or reply.  
In this page, teachers and students can reply the thread by filling the reply box and clicking the reply button.  
If the teacher or student is the creator of the thread, then a hamburger icon will be displayed at the upper right of the thread. When clicked, it will display a menu with 2 options:

1. Edit thread: If clicked it will display the edit thread dialog.
2. Delete thread: If clicked it will display the delete thread dialog.

If the teacher or student is the creator of a reply, then a hamburger icon will be displayed at the upper right of the thread. When clicked, it will display a menu with 2 options:

1. Edit reply: If clicked it will display the edit reply dialog.
2. Delete reply: If clicked it will display the delete reply dialog.
